## What is String Art Generator?

String Art Generator is a free, browser-based tool that transforms any photograph into a string art pattern. It uses a greedy algorithm — powered by OpenCV — to determine the most efficient sequence of thread paths between pins arranged in a circle, recreating your image using only straight lines of thread.

The result is both a digital preview and a numbered pin sequence you can use to actually build the piece on a wooden hoop or canvas.

## How the algorithm works

The generator uses a greedy line-selection algorithm that iterates up to 4,000 times. At each step it evaluates every possible line from the current pin to another pin (skipping pins that are too close or were used recently) and selects the line that best reduces the current error between the target image and the string art rendered so far.

<Note>
  For best results, use a high-contrast, close-up portrait or graphic. Busy backgrounds or low-contrast images produce less defined results.
</Note>
